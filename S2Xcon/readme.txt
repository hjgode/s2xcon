S2Xcon
	application to automate configuration barcodes for ITC devices
	creates pdf with name of input file

run
	s2xcon E:\WLAN_support.xml
		creates E:\WLAN_support.pdf
		and E:\WLAN_support.log

needed runtimes
	S2pHelper.dll
	S2X.dll
	Syncfusion.Compression.Base.dll
	Syncfusion.Core.dll
	Syncfusion.Pdf.Base.dll
	zlib1.dll
